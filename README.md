## Building

### Preliminary 

To start, the repository has to be cloned:

```bash
$ git clone 
$ cd 
```

### Building running the test cases

To create build folder and run tests:

```bash
$ cmake -S . -B build
$ cmake --build build
$ cd build && ctest
```
